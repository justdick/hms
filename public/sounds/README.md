# Vitals Alert Sounds

This directory contains audio files for the vitals monitoring alert system.

## Required Sound Files

### vitals-alert-gentle.mp3
A soft, gentle beep sound for due vitals alerts. Should be:
- Short duration (1-2 seconds)
- Soft tone (around 440-880 Hz)
- Pleasant and non-alarming
- Volume: moderate

### vitals-alert-urgent.mp3
A more prominent tone for overdue vitals alerts. Should be:
- Short duration (1-2 seconds)
- More prominent tone (around 880-1760 Hz)
- Attention-grabbing but not harsh
- Volume: louder than gentle alert

## Generating Sound Files

You can generate these sounds using:
1. Online tone generators (e.g., https://www.szynalski.com/tone-generator/)
2. Audio editing software (Audacity, GarageBand, etc.)
3. Free sound libraries (freesound.org, zapsplat.com)
